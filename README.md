**📊 Profit & Loss Report (PLR) Analysis Dashboard**

This project is inspired by real-world industrial operations where minimizing production loss and identifying faulty equipment is critical for operational efficiency.

**Project Overview**

This project analyzes Production Loss Reports (PLRs) using an interactive dashboard built in Microsoft Power BI.

The focus is on identifying bad actor equipment (frequent tripping sources) and evaluating their impact on plant performance and production loss.

**🎯 Objective**

Identify equipment causing maximum trips (bad actors)

Analyze units responsible for recurring faults

Assess impact on production loss

Support data-driven maintenance decisions

**📊 Key Visualizations**

Plant vs Bad Actors → Highlights plants with highest faulty equipment

Unit vs Bad Actors → Identifies responsible units

Unit vs PLRs → Shows recurring issues across units

Production Loss by Plant → Quantifies plant-wise losses

KPIs → Total PLRs, Total Production Loss, Production Loss per Plant

Bad Actors Analysis → Equipment with highest tripping frequency

**📈 Key Insights**

Few equipment act as major bad actors

Certain units generate higher PLRs

Production loss is concentrated in specific plants

Repeated failures indicate need for preventive maintenance
